Full Cinematic Animation — 12–15s Seamless Loop

START FRAME: A sleek modern laptop, open at 110°, centered on a near-black background. The laptop screen shows a dark code editor with a single line of code: "function choosePath() {}" displayed in soft neon blue monospace font. A blinking cursor sits after the closing brace. Soft screen glow spills onto the keyboard. Perfectly still. Studio lighting.

END FRAME: Same laptop, same position, same framing. The code text on screen is mid-dissolve — characters becoming particles. A continuous neon path line (blue-to-violet gradient, softly glowing) extends from the laptop screen's right edge across the full width of the frame in a smooth Bezier arc.

ANIMATION SEQUENCE — execute precisely in this order:

[0.0s – 0.5s] HOLD: Complete stillness. The laptop screen is on, code visible, cursor blinking at 1Hz. No motion whatsoever. Allow the viewer to read the code line.

[0.5s – 3.5s] TYPING SEQUENCE: The code line types itself character by character, right-to-left erasure → re-typing simulation OR: begin with an empty cursor, type out "function choosePath() {}" one character at a time. Typing speed: deliberate and confident — approximately one character every 0.1–0.15 seconds. Each keystroke registers visually as a very subtle screen flicker (1–2% brightness pulse). The cursor advances with each character. Final character types at ~3.2s. Hold on completed line for 0.3s.

[3.5s – 4.2s] LIFT-OFF INITIATION: The blinking cursor transforms — instead of blinking off, it stretches horizontally into a tiny glowing dash, then a 6px glowing dot. The dot begins a slow, smooth movement to the right along the baseline of the code text. As it moves, it leaves a hair-thin luminous trail behind it (the beginning of the path).

[4.2s – 7.5s] CHARACTER DISSOLUTION: As the moving dot passes over each character from right to left, each glyph:
  — Brightens briefly (0.1s)
  — Fragments into 8–12 micro-particles that orbit momentarily
  — Each particle accelerates outward, following the direction of the growing path
  — The glyph's position on screen fades to empty
The dissolution moves character-by-character, left to right through the word, taking ~3 seconds total. The path line simultaneously grows from the screen's right edge — each dissolved character feeds more luminous particles into the growing neon line.

[7.5s – 10.0s] PATH EXTENSION: The neon path (now fully formed from the dissolved code) extends smoothly beyond the laptop screen, flowing rightward and slightly upward across the hero section in a cinematic Bezier arc. The path moves at a steady, graceful pace — like ink flowing through a very fine nib. The path's tip glows brightest (100%) while the section near the screen settles to ~70% opacity with a soft outer halo.

PARALLAX NOTE: The path moves on a plane slightly in front of the background — a subtle 3D parallax effect of 8–12px suggests depth.

[10.0s – 12.5s] PATH FADE AND RETURN: The path's far tip (right edge) begins to fade — opacity drops from 70% to 0% over ~2 seconds, moving inward toward the laptop screen like the path is being retracted. Simultaneously the path's glow reduces, blur increases slightly. The path retreats fully back into the screen area.

[12.5s – 13.5s] RESET: The laptop screen shows an empty cursor, blinking. The single line begins typing again — seamless loop restart.

CAMERA: Completely locked — zero camera movement throughout. No drift, no zoom, no shake. The only motion is the code typing, dissolution, and path animation.

LIGHTING: No change in scene lighting. The path's glow casts a very faint blue-violet ambient light on the laptop's palm rest area — this should be subtle (3–5% brightness increase only).

STYLE: Premium motion design. The laptop is photorealistic throughout. The code-to-path transformation feels like a real UI effect, not a visual effect. The path is elegant, not flashy. Think: Stripe / Linear / Vercel brand motion — confident, minimal, precise.

TIMING: All easing is ease-in-out cubic. No linear motion. No abrupt transitions.
DURATION: 13–14 seconds, seamless loop.
ASPECT RATIO: 16:9.
FRAME RATE: 60fps preferred, 24fps minimum.
QUALITY: Maximum fidelity. No compression artifacts. Smooth particle rendering.
